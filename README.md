# spring-cloud-zull

First Step \
Build and run example service: \
``cd ./company`` \
``mvn clean install`` \
``mvn spring-boot:run`` \
\
Second Step  \
Build and run Zuul proxy:
``cd ./zuul`` \
``mvn clean install`` \
``mvn spring-boot:run`` \
\
Open your browser: \
``http://localhost:8080/api/company`` \
\
Enjoy it!!!
